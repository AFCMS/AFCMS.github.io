# AFCM's Site

AFCM's personal portfolio website.

## Used Libraries

- [**Preact**](https://preactjs.com)
- [**Vite**](https://vite.dev)
- [**Vite PWA**](https://vite-pwa-org.netlify.app)
- [**TailwindCSS**](https://tailwindcss.com)
- [**React Icons**](https://react-icons.github.io/react-icons)
- [**Oxfmt**](https://oxc.rs/docs/guide/usage/formatter)
- [**PNPM**](https://pnpm.io)

## Usage

### Development Server

```shell
pnpm run start
```

### Production Build

```shell
pnpm run build
```

### Serve Production Build

```shell
pnpm run serve
```

### Rebuild PWA Assets

```shell
pnpm run generate-pwa-assets
```

### Deploy on GitHub Pages (Using GitHub CLI)

```shell
gh workflow run 47919864
```

### Autoformatter

Install the official Oxc VSCode extension and add it as base formatter, enable format on save.

The `oxfmt` npm package needs to be installed locally. (It is in the dependencies)
